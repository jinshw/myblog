# 说明

**本项目为hexo搭建**

```
hexo clean     清除生成文件
hexo g		   生成文件
hexo d		  发布到jinshw.github.io上
hexo serve		启动服务
hexo new <title>  新建文章 

简写
hexo g：hexo generate
hexo c：hexo clean
hexo s：hexo server
hexo d：hexo deploy
```
打开hexo admin页面
localhost:4000/admin
